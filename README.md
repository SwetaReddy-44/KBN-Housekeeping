# KBN Housekeeping Management System

A comprehensive web-based management system designed to streamline daily operations for KBN Housekeeping Services. Built using Python and Flask, this application allows administrators to easily manage bookings, employee attendance, salary calculations, and customer feedback.

## 🚀 Features

*   **Customer Booking Portal:** Customers can easily view available services (e.g., Deep Cleaning, Mopping, Window Cleaning) and book appointments online.
*   **Admin Dashboard:** A secure login portal for administrators to oversee the entire business.
*   **Employee Management:** Track active employees, attendance, and calculate monthly salaries.
*   **Service & Reports Tracking:** Auto-generate reports on total bookings, frequent customers, and top service locations.
*   **Feedback System:** Customers can submit feedback, which admins can review to improve service quality.

## 🛠️ Technologies Used

*   **Backend:** Python, Flask framework
*   **Database:** SQLite (with standard SQL queries)
*   **Frontend:** HTML5, Vanilla CSS, Jinja2 Templating
*   **Design/Architecture:** PlantUML (for system class diagrams)

## 💻 How to Run Locally

1. Clone this repository to your local machine:
    git clone https://github.com/SwetaReddy-44/KBN-Housekeeping.git

2. Navigate to the project directory and run the application:
    python app.py

3. Open your web browser and go to http://127.0.0.1:5000/
